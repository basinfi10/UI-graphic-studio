# UI Graphic Studio Pro v0.9.0

React + TypeScript 기반의 UI 그래픽 워크스페이스입니다.

## v0.9.0 변경점

- 오른쪽 패널의 두 번째 메뉴(도형/아이콘/샘플 UI/컴포넌트/스타일/토큰/정렬/레이어/속성/캔버스/코드)를 중앙 정렬형 메뉴 바로 개선했습니다.
- 기존처럼 오른쪽 끝으로 몰리거나 가로 스크롤에 의존하지 않도록 `flex-wrap + center` 구조로 보정했습니다.
- 외부 CDN 없이 바로 사용할 수 있는 내장 SVG 아이콘 라이브러리를 추가했습니다.
- 기존 텍스트/이모지 아이콘과 별도로, 더 세련된 선형 SVG 아이콘을 캔버스에 추가할 수 있습니다.
- SVG 아이콘은 레이어로 들어가며 색상, 크기, 배경, 그림자, 엠보싱 등 기존 속성 체계를 따릅니다.


### v0.9.0 레이아웃 안정화

- 오른쪽 패널 안에 있던 두 번째 작업 메뉴를 분리해, 상단 중앙의 독립 패널 네비게이션으로 이동했습니다.
- 캔버스 하단 플로팅 컨트롤(새 작업/화면 맞춤/줌/1:1/Pan 0/줌 퍼센트)이 항상 보이도록 레이아웃을 고정했습니다.
- 왼쪽 툴바는 창 높이가 작아도 내부 스크롤로 끝까지 접근할 수 있게 보정했습니다.
- 캔버스 영역은 `min-height:0` 기반 grid 구조로 정리하여 화면 맞춤과 스크롤 영역이 서로 밀리지 않게 했습니다.
- 우측 서브 패널은 탭 메뉴를 숨기고, 선택된 패널 내용만 표시해 하단 메뉴까지 스크롤 가능하게 정리했습니다.

## v0.9.4

이번 버전은 Figma식 도형 생성 흐름을 반영한 도형 작업성 개선판입니다.

- 왼쪽 도형 도구 선택 후 캔버스에서 드래그하여 직접 크기 지정
- 단축키: V 선택, R 사각형, O 원/타원, L 선, A 화살표
- Shift + 드래그: 사각형/원 비율 유지, 선/화살표는 45도 단위 각도 스냅
- Alt + 드래그: 시작점을 중심으로 도형 생성
- 선/화살표 실제 드로잉 지원
- 도형 stroke style: solid / dashed / dotted
- stroke cap: butt / round / square
- stroke join: miter / round / bevel
- Auto Layout Lite: 선택 요소 가로/세로 자동 배치, 12px/24px gap 프리셋
- Boolean 준비 구조: union / subtract / intersect / exclude 메타데이터 저장

Boolean은 아직 실제 path 병합 엔진이 아니라 준비 구조입니다. 다음 단계에서 SVG path boolean 라이브러리 또는 자체 path 연산으로 확장할 수 있습니다.
